1.0本書の目的PURPOSE

積みプラの管理に使えるツールは複数存在する。本仕様書は、模型管理に特化した代表的な 3 サービス（TSUMITSUMI / Tsumina / GUNPLAB）と、 汎用の比較対象として Excel/スプレッドシート を加えた 4 ツールを 10 項目の機能基準 で中立的に評価する。 本書の発行元は TSUMITSUMI 開発元（TSUMI TSUMI）であり、自社サービスの長所のみならず短所も明記する。

  1. 模型管理特化アプリは JAN登録 / 状態管理 はおおむね揃う。差別化点はそれ以外の機能にある。
  2. 価格自動取得（TSUMITSUMI）、クラウド同期（GUNPLAB）、SNSシェア統計（Tsumina）など、各アプリに核となる強みがある。
  3. 選定基準は ① 登録規模、② PC/モバイル/Webの使い分け、③ 必須機能（価格・塗料・同期 etc.） の 3 軸でほぼ決まる。

3.0各ツール仕様シートSPEC SHEETS

各ツールの強み・弱み・推奨される利用者像を仕様シート形式で記述する。

TOOL-001 TSUMITSUMI
模型管理に特化した Web アプリ。バーコードスキャンで JAN を読み、希望小売価格を自動取得して総額を可視化。 ブラウザ上で動作し、データはローカル端末に保存される（クラウド同期なし）。
PROS / 強み
  • JANスキャンで爆速登録（数秒/件）
  • 希望小売価格の自動取得・総額計算
  • ガンプラ/スケール/フィギュア対応のマスタ約7,000件
  • 登録不要・無料・広告なし（一部協賛除く）
  • モバイル最適化、片手で完結
CONS / 弱み
  • クラウド同期なし、端末故障時はバックアップ必須
  • PC でのバルク編集機能は限定的（モバイル前提）
  • マスタにない海外キット等は手入力
  • localStorage 上限 (約5MB) の制約
RECOMMENDED FOR / 向いている人
積みプラ 30 個以上、モバイル中心、JAN 入りキット中心、すぐ可視化したい
NOT FOR / 向かない人
複数端末同期必須、PC専業、海外マイナーキット中心、超大規模 (1000+) 管理
TOOL-002 Excel / Google Sheets 自作
表計算ソフトでテンプレートを自作する方式。集計・グラフ化の自由度が最も高い反面、登録の手作業負荷が大きい。
PROS / 強み
  • カラム構成・集計式が自由
  • バックアップ・移行が容易（ファイル単位）
  • PC での大量編集・並べ替えに強い
  • Google Sheets ならクラウド同期
CONS / 弱み
  • JANスキャン不可、すべて手入力
  • 価格情報は別途手調査
  • テンプレ設計に初期コスト
  • モバイルで操作しづらい
RECOMMENDED FOR / 向いている人
PC 中心、表計算が得意、独自集計したい、自分の仕組みを作る楽しみがある人
NOT FOR / 向かない人
手入力が苦痛、モバイル中心、技術仕様を考えるのが面倒な人
TOOL-003 Tsumina（ツミナ）
個人開発者（Hironori Sakata 氏）による積みプラ管理アプリ。iOS / Android のネイティブアプリで、 バーコードスキャン登録・「積む／製作中／完成／欲しい」の 4 ステータス管理・タグ・統計と SNS シェア機能が好評。 無料枠は登録 100 件まで、それを超えるとプレミアム（¥480/月）が必要。
→ tsumina.app
PROS / 強み
  • ネイティブアプリの操作感、iOS UX が洗練
  • バーコードまとめスキャン
  • 「欲しい」リストを含む 4 ステータス
  • 統計レポート + SNS シェア機能
  • v5.22.0 で「ガンダムデータベース」追加
  • アカウント登録不要
CONS / 弱み
  • 無料は 100 件まで、超えるとサブスク必須
  • 買い切り版なし（レビューでも要望多数）
  • 希望小売価格の自動取得は公式説明なし（?）
  • データは端末ローカル、複数デバイス同期なし
  • 無料版に広告あり
RECOMMENDED FOR / 向いている人
iPhone ユーザー、登録 100 件未満、統計やシェアを楽しみたいライト〜中規模の積み師
NOT FOR / 向かない人
買い切り派、月額課金が嫌な人、100 件超の大量管理、複数端末で同期したい人
TOOL-004 GUNPLAB（ガンプラボ）
ガンプラ特化の Web アプリ。PC / スマホ両対応で、Google アカウントでログインしてクラウド管理。 「積みプラ／ビルド中／完成／欲しい」の 4 ステータスに加え、グレード・作品別の絞り込み、 そして他にはない 塗料管理機能（メーカー横断 DB + カラーチップ表示）を持つのが最大の差別化点。
→ gunplab.fun
PROS / 強み
  • 塗料管理（メーカー DB + カラーチップ）はこのカテゴリで唯一
  • Web 完結で PC / スマホ両用、自動同期
  • ガンプラ特化のグレード・作品分類 DB
  • 専用 URL でコレクション共有可能
  • 無料、登録数の上限なし
CONS / 弱み
  • ネイティブアプリなし、Web のみ
  • Google アカウント必須
  • 価格情報はコミュニティ手動更新（自動取得なし）
  • 非公式ファンサイトで運営者は個人
  • 将来的な有料プラン・広告導入が示唆されている
RECOMMENDED FOR / 向いている人
塗料も含めて一元管理したい本格派、PC とスマホを両用したい人、コレクションを公開・共有したい人
NOT FOR / 向かない人
アカウント登録を避けたい人、ネイティブアプリ派、ガンプラ以外（スケールモデル中心など）の人

4.0選定フローチャートDECISION FLOW

3 つの設問に答えれば、向いているツールが絞れる。

Q1 あなたの積みプラはおおよそ何個ですか？
10 個未満 ／ どのツールでも十分。Tsumina の無料枠（100 件）にも余裕。 → ANY
10 〜 100 個 ／ 模型特化アプリの効率が活きる規模。
TSUMITSUMI / Tsumina / GUNPLAB から好みで選択。 → TSUMITSUMI / TSUMINA / GUNPLAB
100 個以上 ／ Tsumina は無料枠超過で月額発生。
無料で大量管理するなら TSUMITSUMI / GUNPLAB / Excel。 → TSUMITSUMI / GUNPLAB / EXCEL
Q2 主にどの環境で使いますか？
PC 中心 ／ Excel/Sheets または Web ベースの GUNPLAB。
Tsumina はネイティブアプリのみ、TSUMITSUMI も主にモバイル想定。 → EXCEL / GUNPLAB
モバイル中心 ／ 片手で完結する TSUMITSUMI / Tsumina。
店頭でその場で登録できる強み。 → TSUMITSUMI / TSUMINA
PC とスマホ両方 ／ クラウド同期の GUNPLAB が唯一の正解。
Excel/Sheets でも可だが手作業多め。 → GUNPLAB
Q3 最も重視する機能は何ですか？
希望小売価格・総額の自動把握 ／ TSUMITSUMI のみ標準対応。
他は手入力／コミュニティ更新。 → TSUMITSUMI
塗料の管理も一緒に ／ GUNPLAB の塗料 DB はこのカテゴリで唯一。 → GUNPLAB
SNS シェア・統計レポート ／ Tsumina の統計機能はモデラーに好評。 → TSUMINA
独自の集計式・自由なテンプレ ／ Excel/Sheets。
ピボット・グラフは表計算ソフトの本領。 → EXCEL

5.0結論CONCLUSION

模型管理特化の 3 アプリ（TSUMITSUMI / Tsumina / GUNPLAB）は基本機能（JAN登録・状態管理）はおおむね揃う。 決定的な差は 核機能の方向性にある：

  • TSUMITSUMI ＝ 希望小売価格の自動取得・総額可視化（金額把握が最強）
  • Tsumina ＝ ネイティブアプリの操作感・統計レポート・SNS シェア（ライト〜中規模で楽しむ）
  • GUNPLAB ＝ Web 完結・クラウド同期・塗料管理（本格モデラー向け）

Excel/Sheets はこれら特化アプリと役割が違うので、独自の集計をしたい人や PC バルク編集中心の人のみに推奨される。 本書はあくまで「自分の優先順位を整理するための判断材料」である。

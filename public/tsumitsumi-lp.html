Features

FULL
FEATURES

01
登録
  • バーコードスキャンで即時登録
  • キット名入力でマスタ候補を表示（グレード・スケール・キット名）
  • グレード別リストからの一括登録
  • 希望小売価格を自動取得・表示
  • 箱の写真・完成品の写真を保存
  • 個数・購入日・評価・状態・メモを記録
02
管理 / 編集
  • 登録済みキットの詳細表示・編集・削除
  • 一括編集モード（状態・シリーズ・スケールを一括設定）
  • タグの付与・削除（タップで選択、赤丸 x で削除）
  • タグを一括でキットに追加・削除
  • 一括削除
03
検索 / 絞り込み
  • キーワード検索
  • スケール・シリーズ・評価・状態でフィルタリング
  • タグによる AND 絞り込み
  • 名前順・登録順・購入日順・手動順でソート
  • 昇順・降順の切り替え
04
表示
  • 詳細リスト表示・サムネイルグリッド表示の切り替え
  • カードにシリーズ・スケール・タグを表示
  • 積みプラ数・完成数・総数を個数合計で表示
  • 積みプラの総額を見て、現実と向き合う
  • 登録数に応じた称号システム
05
シェア / バックアップ
  • 積みプラ数・選択キットを X に投稿
  • 「譲りたい」ポスト機能
  • JSON エクスポート・インポートでバックアップ

How To Use

3 STEPS
TO START

1
ホーム画面に追加する
Safari や Chrome でサイトを開き、「ホーム画面に追加」を選択。インストール不要でアプリとして使えます。
2
バーコードをスキャンする
プラモデルの箱に印刷されたバーコードをカメラで読み取るだけ。マスタから商品情報が自動で入力されます。
3
タグ・状態を整理する
「未開封」「プレバン限定」などのタグを付けて分類。一括編集モードで複数のキットをまとめて整理できます。

TSUMITSUMI 完全マニュアル

積みプラを楽しく管理するための機能と使い方を、全部このページにまとめました。
気になるところから読んでください。

はじめに

TSUMITSUMI（ツミツミ）は、積みプラの管理に特化した無料のWebアプリです。
「いま家に何があるか」「いくらぶん積んでるのか」「どれを完成させたか」を、サッと記録・確認できます。

こんな人に向いています

  • 気がつくと積みプラが増えていて、何を持っているか把握しきれない
  • 同じキットを買ってしまった経験がある
  • 積みプラの総額をなんとなく知っておきたい（あるいは知りたくない）
  • 完成したキットを記録して達成感を味わいたい

主な特徴

  • 📷 バーコードスキャンで一瞬登録
  • 🔍 名前・スケール・タグで自由に検索
  • ✓ 完成マークでビフォーアフターを記録
  • 💾 ブラウザに保存。アカウント登録不要
  • 📤 バックアップ書き出し / 別端末への引き継ぎ対応

最初のステップ

使い始めはとてもシンプルです。アカウント登録もインストールも不要です。

  1. TSUMITSUMI を開く
    ブラウザで tsumitsumi.vercel.app にアクセスします。
  2. ホーム画面に追加（推奨）
    スマホのブラウザの共有メニューから「ホーム画面に追加」を選ぶと、アプリのように起動できます。
  3. 最初のキットを登録
    右上の ＋ 追加 ボタンから登録できます。
💡

データはお使いのブラウザに保存されます。別の端末や別のブラウザで使うときは、後述のバックアップ機能で引き継いでください。

キットを登録する

キットの登録方法は3つあります。状況に応じて使い分けてください。

① バーコードスキャン（一番ラク）

新品で箱があるなら、これが最速です。バーコードスキャンの章で詳しく説明します。

② 商品名で検索して登録

箱がもう手元にない／中古品でバーコードが読めない場合に便利。

  1. ＋ 追加 をタップ
  2. 「商品名で検索」欄にキット名を入力（例：「RG νガンダム」）
  3. 候補から正しい商品をタップ
  4. 必要に応じて個数や購入金額を編集して保存

③ 手入力で登録

ガレージキット、ジャンク扱いの福袋、自分で改造したキットなど、検索に出てこないものはこちら。

  1. ＋ 追加 をタップ
  2. 「手入力」を選択
  3. 商品名・スケール・価格・購入店などを入力
  4. 必要なら写真を追加して保存

登録できる項目

項目説明

シリーズ例：機動戦士ガンダム / フレームアームズ・ガール

バーコードスキャン

箱の裏のバーコードをカメラで読み取って一発登録します。プラモデル管理アプリでも一番使う機能です。

使い方

  1. ＋ 追加 をタップ
  2. 「バーコードスキャン」をタップ（初回はカメラ使用許可を出してください）
  3. 箱のバーコードに枠を合わせる
  4. 読み取れると自動的に商品情報が表示されます
  5. 個数・購入価格などを調整して保存
📷

うまく読めないときのコツ
・バーコードと10〜15cmぐらい離す
・明るい場所で読み取る
・斜めではなく真正面から
・反射するときは少し角度をずらす

商品情報が出てこなかったら？

JANコードはツミツミのデータベースに登録されている必要があります。
未登録の商品でも、バーコード番号は自動で入りますので、商品名などを手入力で補完して保存してください。
あなたが登録した情報は、他のユーザーには共有されません。

連続スキャンモード

大量のキットをまとめて登録したいときに便利な機能です。
箱を1つずつ取り出してスキャンするのではなく、ベルトコンベアのようにカメラを開いたまま連続でバーコードを読み込めます。

使い方

  1. バーコードスキャン画面で 🔄 連続ON をタップ
  2. 1個目のバーコードを読み取る → キューに自動追加（カメラは閉じない）
  3. 「次のバーコードをスキャン...」と表示されるので、続けて2個目を読み取る
  4. 必要な数を読み込んだら、画面下の 完了 をタップ
  5. キューに溜まったキットを一覧で確認 → 一括で保存
ℹ️

連続モードはOFFにすると1回ずつ確実に登録できるモードに戻ります。
状況によって切り替えてお使いください。

こんな場面で便利

  • 引っ越しや整理で、押入れの積みプラをまとめて棚卸し
  • イベントやセールで複数買いした帰り道
  • 友達や家族のコレクションを一気に登録したいとき

検索・絞り込み

登録したキットが増えてくると、目的のキットを探す機能が役に立ちます。

キーワード検索

画面上部の検索ボックスに、商品名やシリーズ名の一部を入力すると、ヒットしたキットだけが表示されます。

スケール・グレードでフィルター

スケール選択ドロップダウンから絞り込めます。

  • 1/144、1/100、1/72、1/60、1/48、1/32、1/24
  • HG、RG、MG、RE/100、MGSD、MGEX、PG、SD
  • フルメカニクス、UNLEASHED
  • その他
  • デカール（水転写デカール等の補助パーツ管理用）

タグでフィルター

自分で付けたタグでも絞り込めます。
例：「塗装中」「制作待ち」「子どもと作る」「コンペ用」など。

表示モードの切り替え

キット一覧はグリッドビュー（写真メイン）とリストビュー（情報メイン）を切り替えられます。
気分やシチュエーションに合わせてお選びください。

キットの詳細・編集

詳細画面の開き方

一覧でキットをタップすると詳細画面が開きます。
写真、商品情報、メモ、タグなどを確認できます。

編集する

詳細画面の 編集 ボタンから、内容を書き換えて保存できます。

削除する

詳細画面の 削除 ボタンで、そのキットを一覧から削除します。
削除すると元に戻せませんのでご注意ください。

⚠️

誤って削除してしまった場合、最近のバックアップがあれば復元できます。バックアップの章を参照してください。

完成マークをつける

キットを完成させたら、忘れずに完成マークをつけましょう。
積みから「作った」になる瞬間です。

つけ方

  1. キットの詳細画面を開く
  2. 完成 ボタンをタップ
  3. 一覧の該当キットに ✓ 完成済み バッジが表示されます

完成マークでできること

  • 一覧で完成したキットがひと目で分かる
  • 「達成感」がたまっていくモチベーションに
  • 未完成だけ表示する絞り込みもいずれ拡充予定

タグ機能

タグは自分で自由に決められる分類ラベルです。
シリーズやスケールでは表現しきれない「自分なりのカテゴリ」を作るのに使います。

タグの使い方の例

  • 制作ステータス：「塗装中」「素組み完成」「ディテールアップ中」
  • 用途：「子どもと作る」「コンペ提出予定」「練習用」
  • 場所：「クローゼット」「実家」「会社の机の中」
  • 気分：「いつか作る」「すぐ作りたい」「永久保存」

タグの追加・削除

キット編集画面でタグ欄に文字を入力 →「追加」ボタンで登録できます。
不要になったタグはタグ表示の × で削除できます。

一括登録

キーワードで商品をまとめて検索して、 チェックを入れたものを一気に登録できる機能です。
引っ越しや棚卸し、 同じシリーズをまとめて登録したいときに便利です。

使い方

  1. 右下の ≡ 一括登録 ボタンをタップ
  2. 検索ボックスにキーワードを入力（例：「バンダイ HG ガンダム」）
  3. 🔍 検索 をタップ → 候補一覧が表示されます
  4. 登録したい商品にチェックを入れる
  5. 下部の「✓ 〇件をまとめて登録」ボタンで一気に登録完了

AND検索について

キーワードをスペース区切りで複数入れると、すべての単語を含む商品だけが表示されます（AND検索）。

  • HG ガンダム → 「HG」と「ガンダム」両方を含む商品
  • コトブキヤ メガミ → コトブキヤのメガミデバイスシリーズだけ絞り込み
  • 1/144 シャア → スケール 1/144 でシャア関連だけ
💡

キーワードを増やすほど絞り込まれます。
最初は1〜2語でゆるく検索 → ヒット数が多すぎたら語を足して絞り込むのがおすすめです。

ℹ️

一括登録で登録される情報は商品名・画像のみです。
購入日・価格・状態・タグなどの詳細は、 登録後に各キットの編集画面から個別に追加してください。

合計金額・統計

登録した全キットの合計金額・総個数を、ホーム画面で確認できます。

  • 合計金額：購入価格の合計（手入力した価格があればそれが優先）
  • 総個数：登録した全キットの個数を合計
  • 完成数：完成マークがついているキットの数
😱

合計金額の数字を見て震えるかどうかは、あなた次第です。
怖くなったら、表示を非表示に切り替える機能もあります。

SNSシェア

登録した数や合計金額、完成キットなどを、X（旧Twitter）でシェアできます。
積み師仲間と語るネタとしてどうぞ。

シェアの仕方

  1. ホーム画面のシェアアイコン 𝕏 をタップ
  2. シェアしたい内容（合計金額・個数・完成数など）を選択
  3. X が起動して投稿画面に切り替わるので、そのままポスト

シェアされる情報

個別のキット情報や購入店などのプライベートな情報はシェアされません。
合計値・個数・完成数など、集計された情報のみが含まれます。

バックアップと機種変更

TSUMITSUMI のデータはあなたの端末のブラウザに保存されます。
機種変更・ブラウザの引っ越し・万が一のデータ消失に備えて、定期的なバックアップをおすすめします。

バックアップを書き出す

  1. メニューから「バックアップ」を選択
  2. バックアップを書き出す をタップ
  3. JSON形式のファイルがダウンロードされます
  4. このファイルをクラウドストレージや自分宛メールなど、安全な場所に保存

バックアップから復元する

  1. 新しい端末／ブラウザで TSUMITSUMI を開く
  2. メニューから「バックアップ」を選択
  3. バックアップから復元 をタップ
  4. 保存しておいた JSON ファイルを選ぶ
  5. 確認画面で「復元」を実行 → データが移行されます
⚠️

復元すると現在のデータは上書きされます。
「いまの端末のデータ」と「復元するデータ」を混ぜることはできません。
必要に応じて、復元前にいまのデータを書き出しておきましょう。

こんなタイミングで取っておきたい

  • ある程度キットを登録できたとき（区切りごとに）
  • 機種変更やブラウザ移行の前
  • OSアップデート前
  • 大事なデータを編集する前

保存容量について

TSUMITSUMI はブラウザの保存領域を使っています。
多くのキットを写真付きで登録すると、容量が増えていきます。

容量の確認

ヘルプ画面の「保存容量」セクションに、現在の使用量と上限の目安が表示されます。

容量を圧迫しないコツ

  • 写真は1キットにつき1枚を目安に
  • 解像度の高すぎる写真はリサイズしてから登録
  • 不要になったキット情報は削除
  • 定期的にバックアップを書き出して、本体データはスッキリ保つ
⚠️

容量の上限に近づくと、警告が表示されます。
上限を超えると新しいキットの登録ができなくなりますので、早めに対処してください。

困ったときは

カメラが起動しない

  • ブラウザのカメラ使用許可を確認してください
  • iPhone の場合：設定 → Safari → カメラ → 「許可」
  • Android の場合：設定 → アプリ → ブラウザ → 権限 → カメラ
  • HTTPS でアクセスしている必要があります（tsumitsumi.vercel.appはOK）

バーコードが読み取れない

  • 明るい場所で、10〜15cm離して、真正面から
  • 反射する場合は少し角度をずらす
  • 古い本やシール越しのバーコードは認識率が落ちます
  • どうしても読めなければ、バーコード番号を直接入力する手もあります

データが消えてしまった

  • ブラウザのキャッシュ／履歴クリアでデータが消えることがあります
  • バックアップを保存していれば、そこから復元できます
  • 定期的なバックアップ書き出しを習慣にしてください

同じキットを2回登録してしまった

  • 不要なほうを詳細画面から削除してください
  • 個数管理として残すなら、片方の個数を増やして片方を削除

商品検索でヒットしない

  • 検索キーワードを変えてみる（型番・カタカナ・ローマ字など）
  • マスタDBに未登録の場合は、手入力で登録できます
  • ガレージキットや中古福袋などは手入力が基本です

更新履歴の見かた

TSUMITSUMI は継続的にアップデートされています。新機能や改善点はヘルプ画面の「更新履歴」で確認できます。

確認方法

  1. アプリのヘルプ画面（❓アイコン）を開く
  2. 「更新履歴」セクションに最新3件のアップデートが表示されます
  3. 「すべて見る」をタップすると、これまでの全アップデートが確認できます

新機能の通知について

最新版には NEW バッジが付きます。
アプリを開いたタイミングで自動的に最新版に更新されますので、特別な操作は不要です。

それでは、よい積みプラライフを！

分からないことがあれば、ヘルプ画面のフィードバックボタンからお問い合わせください。
みなさんのコレクションが、いつか積みから「作った」に変わる日まで、TSUMITSUMI が估走します。
